# Hub de Projetos em Dados

Este repositório contém meu hub de projetos em dados, desenvolvido para organizar e apresentar projetos relacionados a Data Science, Analytics, Business Intelligence, SQL e Python.

Essa página é o meu portfólio e reune projetos práticos, estudos, dashboards e aplicações orientadas a dados.

## Sobre o projeto

O hub foi desenvolvido com o objetivo de centralizar meus principais projetos e facilitar o acesso a dashboards, repositórios e materiais relacionados ao meu desenvolvimento na área de dados.

## Tecnologias utilizadas

- CSS3
- JavaScript
- Power BI
- DAX
- Power Query
- Modelagem de Dados
